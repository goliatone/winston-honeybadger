# wiston-honeybadger

Honeybadger transport for winston

## Getting Started
Install the module with: `npm install wiston-honeybadger`

```javascript
var wiston_honeybadger = require('wiston-honeybadger');
wiston_honeybadger.awesome(); // "awesome"
```
